# The Unemployment Pandemic

**Navigating meaning, stability, and peace in the AI era — because you are not alone.**

---

## 🔍 The Problem

The world is entering an unprecedented era of workforce disruption. Artificial intelligence, automation, and rapid technological change are reshaping entire industries — and millions of people are caught in the crossfire.

- **Mass displacement is accelerating.** Reports estimate that up to 300 million jobs globally could be affected by AI within the next decade.
- **Emotional preparedness is near zero.** Most people have no framework for processing the identity crisis, anxiety, and loss of purpose that accompanies job loss.
- **Structured guidance barely exists.** Traditional career services focus on résumés and job boards — not on the human side of unemployment.

The result? A silent pandemic of isolation, fear, and lost direction.

---

## 💡 The Solution

The Unemployment Pandemic is a platform built to meet people where they are — emotionally, practically, and strategically.

We don't just help people find the next job. We help them find themselves in a world that's changing faster than anyone expected.

- **Understand the landscape** — See the bigger picture of AI disruption with curated news, government policies, and expert analysis.
- **Process what you're feeling** — Guided content that validates your emotions and helps you move through grief, anger, and uncertainty.
- **Build real strategies** — Frameworks for financial stability, skill adaptation, and lifestyle restructuring during transition.
- **Connect with others** — A global community of people navigating the same storm, sharing stories, and lifting each other up.

---

## 🎯 What Makes This Different

Most platforms treat unemployment as a logistics problem — update your résumé, apply to more jobs, network harder.

We treat it as a **human problem**.

- **Emotional + Practical** — We address the full spectrum: from processing grief to building a financial runway.
- **Meaning over metrics** — We help people find purpose, not just paychecks.
- **Future-ready mindset** — Instead of competing with AI, we help people adapt alongside it.
- **No stigma** — Unemployment is not a personal failure. It's a systemic shift. We normalize the conversation.

## 🚀 Getting Started

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project
cd the-unemployment-pandemic

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`.
